# The Practice Sandbox

Welcome! This folder is a **safe place to practice**. Nothing here matters — break it,
change it, delete lines, start over. That's the point.

## What's here

- **`hello.txt`** — a throwaway file you'll edit in the exercises.
- **`exercises.md`** — four guided exercises, from your first commit to using Copilot CLI.

## How to start

1. Make sure you've cloned this repo and you're in the `practice` folder:

   ```bash
   git clone https://github.com/eriic-builds/git-learning-lab.git
   cd git-learning-lab/practice
   ```

2. Open **`exercises.md`** and follow it top to bottom.

3. If anything feels stuck, run `git status` — it almost always tells you what to do next.

> Tip: keep the [live guide](https://eriic-builds.github.io/git-learning-lab/) open in a
> browser tab while you practice, so the explanation and the doing sit side by side.
